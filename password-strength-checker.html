Add PassguardAI

    Open extension store

    Browsers block silent installs from websites. After you publish PassguardAI, set store URLs in code — until then use Developer Mode → Load unpacked from your extension folder.

    PassguardAI app

    INSTALL · SAME UI AS WEB

    Put PassguardAI on your home screen — works offline after first load when served over HTTPS or localhost.

    On iPhone / iPad: tap Share → Add to Home Screen. Apple does not expose one-tap install like Android Chrome.

    Chrome / Edge (Android): open the browser menu (⋮) → Install app or Add to Home screen.

    Google Play App Store

    Native Play Store / App Store builds need a wrapped shell (e.g. Capacitor). Paste listing URLs into PASSGUARD_APP_STORE_URLS when ready — until then use install / Add to Home Screen.
